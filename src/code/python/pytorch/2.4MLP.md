import torch.nn as nn

## 定义模型

不同于 softmax 和简单的线性回归，多层感知机多了隐藏层，由于层数的增多，需要在不同层之间定义激活函数，否则多个线性层最终能合并成一个线性层。

定义隐藏层权重矩阵为`784x256`，定义输出层权重矩阵为`256x10`

```python
num_inputs, num_outputs, num_hidden = 784, 10, 256

net = nn.Sequential(
    d2l.FlattenLayer(),
    nn.Linear(num_inputs, num_hidden),
    nn.ReLU(),
    nn.Linear(num_hidden, num_outputs),
)
```

## 初始化模型参数

```python
for parameter in net.parameters():
    init.normal_(parameter, 0, 0.01)
```

## 读取数据、定义损失函数和优化器并训练模型

这里为了优化代码，直接调用了 `d2lzh` 包中的`train_ch3`函数。

```python
batch_size = 256
train_iter, test_iter = d2l.load_data_fashion_mnist(batch_size)
loss = nn.CrossEntropyLoss()
optimizer = torch.optim.SGD(net.parameters(), lr=0.1)
num_epochs = 5
d2l.train_ch3(net, train_iter, test_iter, loss, num_epochs, batch_size, None, None, optimizer)
```
